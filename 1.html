O que é JavaScript?

a) O JavaScript foi criado para tornar as páginas mais interativas, permitindo ações que o HTML sozinho não faz. Ele trouxe movimento e respostas rápidas dentro do navegador.

b) A execução pode acontecer no navegador ou no servidor. No navegador, o código roda direto no dispositivo do usuário e responde rapidamente às ações na tela. No servidor, ele é processado em um ambiente separado e lida com informações e dados mais sensíveis.

c) Os navegadores usam o sistema de sandbox, que é um espaço isolado onde o JavaScript funciona. Isso evita que o código tenha acesso a partes internas do computador, deixando o uso mais seguro.

d) Alguns objetos globais são o document, usado para mexer no HTML; o console, que serve para exibir mensagens para o desenvolvedor; e o alert, que mostra avisos na tela do usuário.
